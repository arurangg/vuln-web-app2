Your Mission

This application is intentionally insecure. Your goal is to identify, exploit, and then understand how to remediate each of the deliberately introduced vulnerabilities below. Read the source, attack the running app, and analyze the root cause of every flaw.

Vulnerabilities to Discover

SQLi

SQL Injection

Login and signup build SQL via string concatenation, allowing authentication bypass with crafted input.

XSS

Stored XSS

A username is rendered on the dashboard without escaping, persisting injected scripts for every viewer.

XSS

Reflected XSS

The search endpoint reflects the query parameter into the response without escaping.

Session

Session Hijacking

A weak, hardcoded session secret makes session cookies forgeable and reusable.

Crypto

Weak Password Storage

Passwords are hashed with unsalted MD5, reversible via rainbow tables.

Exposed

Exposed Database

The database file is downloadable over HTTP with no authentication required.

Brute

No Rate Limiting

No throttling on any endpoint enables unlimited automated credential guessing.

CSRF

Cross-Site Request Forgery

Forms carry no CSRF tokens, so state-changing requests can be forged from other origins.
